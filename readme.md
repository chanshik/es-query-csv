## Save query result of ElasticSearch to CSV

### Arguments:
```
  -h, --help            show this help message and exit
  -u ES_HOST            ElasticSearch URL
  -j JSON               JSON file for Query DSL
  -o OUTPUT             Output CSV file
  -i INDEX              Index prefix
  -f FIELD              Selected result field to save
  -m MAX_COUNT          Maximum count of output records
  -s USE_SCRIPT_FIELDS  Get results from script_fields
  -d DELIMITER          Delimiter
```
